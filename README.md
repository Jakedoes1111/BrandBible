### Prerequisites
- Node.js 16+
- Gemini API key ([Get one here](https://aistudio.google.com/app/apikey))

### Installation

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Set up your API key:**
   Edit `.env.local` and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

3. **Run the app:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   ```
   http://localhost:3000
   ```
